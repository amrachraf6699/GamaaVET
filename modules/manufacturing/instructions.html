Manufacturing Workflow Guide

This page explains how the GammaVet manufacturing module stages orders, generates formulas from your existing product catalog, and produces the Excel/PDF handoffs that travel with the work.

1. What you set up in “Create Manufacturing Order”
  • Select a customer/provider. The list of formulas is filtered to the ones they already have, or you can build a new formula by picking catalog products for each component.
  • Each component row now requires selecting a catalog product so traceability stays consistent with inventory.
  • You can assign priorities, batch size, due date, and notes; every order gets a unique MAN-yyyyMMdd-XXXXXX number.
2. How the seven-stage workflow works
  1. Sourcing & Procurement. Confirm suppliers, request samples, and create POs. Document approved lead times.
  2. Receipt & Quality Check. Receive materials, verify quantities, log QA inspections, and release them to production.
  3. Preparation & Mixing. Blend the formula, record ratios/temperatures, and stage the blended mix for QA.
  4. Quality Validation. Execute in-process tests, adjust parameters, and verify compliance before packaging.
  5. Packaging & Labeling. Package the batch, print labels, and stage it per delivery instructions.
  6. Dispatch Prep. Ready the transportation, documents (customs, certificates), and special handling instructions.
  7. Delivery & Client Handover. Release the shipment, attach final delivery notes, and capture any outstanding follow-up.
3. Excel / PDF handoff documents

Every time a step is saved, the module writes:

  • An Excel-compatible (.xls) snapshot containing the order number, provider, priority, instructions, and component table.
  • A PDF rendered via TCPDF with the same details plus the step-specific reminders for the next team.

Files are stored under assets/uploads/manufacturing/{ORDER_NUMBER}/{STEP_KEY}/ and linked from the order detail page. Use the download buttons to hand over the latest files.

4. Tips for users
  • Work forwards through the steps. The next step will show as “pending” until the previous one is completed, ensuring linear accountability.
  • Always attach notes before clicking “Save step”. That text appears in both documents and keeps the next hand-off informed.
  • Use the “Workflow timeline” card on the order view to see which step is current, how many files exist per step, and the descriptions of each phase.
